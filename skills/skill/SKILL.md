# Skill

技能管理命令。

## Commands

### 搜索技能

```bash
node dist/cli/index.js skill search <关键词>
```

### 安装技能

```bash
node dist/cli/index.js skill install <技能名>
```

### 列出已安装技能

```bash
node dist/cli/index.js skill list
```

### 查看技能详情

```bash
node dist/cli/index.js skill info <技能名>
```

### 卸载技能

```bash
node dist/cli/index.js skill uninstall <技能名>
```
